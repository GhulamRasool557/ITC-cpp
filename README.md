# ITC C++ Practice Repository

## Overview

A collection of beginner-friendly C++ programs for learning basic programming concepts.

## Purpose

* Build strong fundamentals in C++
* Practice core ITC concepts
* Improve problem-solving skills

## Structure

* **HelloWorld** – Basic programs
* **Integer Tasks** – Conditions and calculations
* **Array Tasks** – Basic array problems
* **Character Array Tasks** – Character arrays (no string)
* **Patterns** – Pattern printing using loops
* **ITC Projects** – Simple beginner projects

## Approach

* All code written inside `main()`
* No functions used
* No `string` library
* Focus on loops, conditions, and arrays
* Simple and readable code

## Target

* ITC students
* Absolute beginners in C++

## Goal

Strengthen programming basics and logical thinking through practice.
